## Install

```sh
npm i
```

## How to run the code

```sh
npm start
```

## Architecture
🎥 **FrontEnd**
* React Leaflet
* Class functions 
* Simple Fetch for Ip Coords (cant fetch directly since Cors Privacy)


💻 **BackEnd**
* Use Mongo Atlas (for an easy and fast search besides saving all Coords and Messages)
